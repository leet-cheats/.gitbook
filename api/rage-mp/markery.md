# Маркеры

### Создать маркер

```typescript
mp.markers.new(type: number, position: mp.Vector3, scale: number,
    {
        direction: mp.Vector3,
        rotation: mp.Vector3,
        color: [number, number, number, number],
        visible: boolean,
        dimension: number
    }): mp.Marker
```
